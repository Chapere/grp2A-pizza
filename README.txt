# Pizza-WebApp

Nachdem Sie das Projekt geclont haben, wechseln Sie auf der Kommandozeile in das
Verzeichnis des Projektes.

Das Projekt ist mit der aktuellsten Scala-Version erstellt (2.11.1) und benötigt
somit die Scala IDE in der Version 4.0.0 oder höher.

Sollten Sie mit der 3.0.3 auf den Laborrechnern arbeiten, tauschen Sie die
`build.sbt` und die `plugins.sbt` mit den in Moodle zur Verfügung gestellten aus,
vor Sie weiter machen.

Geben Sie

    activator eclipse

ein. Starten Sie anschließend die Scala IDE und importieren Sie das Projekt.

Starten Sie auf der Kommandozeile die Activator-Konsole mit dem Kommando

    activator

und geben Sie anschließend

    ~run

ein um bei jedem Speichern der Dateien die Web-Applikation neu zu compilieren.

Sehen Sie sich die Web-Applikation im Browser an.

Sehen Sie sich die folgenden Commits an um zu sehen, wie die Web-Applikation
Schritt für Schritt umgebaut wurde.
